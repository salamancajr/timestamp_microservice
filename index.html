Time Stamp Micro-Service

Hello!

Add a date to the end of the web address to get the Unix timestamp and the natural language form of that date.

An example of this might be the following:

https://timestampnode1.herokuapp.com/May-5-2018,

or

https://timestampnode1.herokuapp.com/1525474800.

The output will look something like this:

{ unix: 1525474800, natural: "May 5, 2018" }
